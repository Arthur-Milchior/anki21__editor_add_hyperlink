This add-on adds a dialog to enter hyperlinks (see screenshot below) and some context menu entries for handling hyperlinks. The shortcuts to insert a hyperlink (default Ctrl+Shift+H) and unlink (Ctrl+Shift+Alt+H) can be configured using the configuration dialogue of 2.1: Tools>Add-ons, select the add-on and click on "Config". Use this at your own risk. This is a partial fork of the add-on Power Format Pack (Copyright 2014-2017 Stefan van den Akker) because it won't be ported to 2.1 (as of 2018-11). I just extracted the hyperlink function into a standalone version that works in 2.1. Also see the original description at Power Format Pack. For some uses cases (links to the same site that have a similar url structure) this built-in feature is better: https://apps.ankiweb.net/docs/manual.html#dictionary-links The source code is available on github.
